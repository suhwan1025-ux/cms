- **AI 서버:** Python 3.12 + FastAPI + Uvicorn
- **백엔드 서버:** Node.js + Express
- **데이터베이스:** PostgreSQL 15

- **LLM:** Ollama + llama3.1:8b

- **API 통신:** Axios
